# Set Up Mac

> 恭喜您入手一台新的Mac！配置合适的开发环境与下载必要的软件是至关重要的。  
> 在本篇指南中，我们将带您一步步完成这个流程，确保您的Mac能够迅速投入使用之中。
## 开发工具包
> 在Mac上安装Xcode的命令行开发工具。这些工具包括编译器、Git以及其他必要的开发工具
```bash copy
xcode-select --install
```
## Homebrew
[Homebrew](https://brew.sh/)是Mac上非常受欢迎的包管理器，能帮助您轻松安装各种开发工具。

```bash copy
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
